# Entry for: UPV KomsaiHack 2026 — *Risk Ready*

## Project: **Tabang**

**Tabang** is a web-based mobile application designed to support communities in **Aklan** during flood-related emergencies. The platform enables residents to report flood incidents, request assistance, and access verified emergency hotlines through a single, community-centered system.

### Key Features
- **Flood Reporting** — Users can submit real-time flood reports to help monitor affected areas.
- **Help Requests** — Individuals can request assistance during emergencies.
- **Hotline Directory** — Centralized access to local emergency contact numbers.
- **Hotline Effectiveness Reviews** — Users can provide feedback and reviews on whether hotlines responded effectively, helping improve reliability and public trust.
- **Responder Dashboard** — Emergency responders have a dedicated homepage where they can view local reports and assistance requests within Aklan for faster coordination and response.
- **Community-Focused** — Built specifically to serve local communities within Aklan.
